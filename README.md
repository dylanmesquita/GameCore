# 🎮 GameCore

GameCore é uma plataforma para gerenciamento e descoberta de jogos, desenvolvida com foco em performance, organização e escalabilidade.

## ✨ Funcionalidades

- Catálogo de jogos
- Busca e filtragem
- Interface responsiva
- Sistema de categorias
- Integração com banco de dados MongoDB
- Arquitetura moderna baseada em React e Next.js

## 🛠️ Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- MongoDB
- Node.js

## 🚀 Instalação

Clone o repositório:

```bash
git clone https://github.com/dylanmesquita/GameCore.git
```

Entre na pasta:

```bash
cd GameCore
```

Instale as dependências:

```bash
npm install
```

Configure as variáveis de ambiente:

```env
MONGODB_URI=sua_string_de_conexao
```

Inicie o servidor:

```bash
npm run dev
```

A aplicação estará disponível em:

```
http://localhost:3000
```

## 📦 Scripts

```bash
npm run dev      # Ambiente de desenvolvimento
npm run build    # Build de produção
npm run start    # Executar versão de produção
npm run lint     # Verificação de código
```

## 🎯 Objetivo

O objetivo do GameCore é fornecer uma base sólida para gerenciamento, exibição e organização de informações sobre jogos, permitindo futuras expansões como autenticação, avaliações, listas personalizadas e integração com APIs externas.
